# 02. Node.js와 npm

Assembled by GimunLee

# 개요

Angular는 간단한 명령어로 Angular 프로젝트의 스캐폴딩(scaffolding)을 생헝, 실행, 빌드 할 수 있으며, 다양한 구성요소를 선별적으로 추가할 수 있는 커맨드 라인 인터페이스인 Angular CLI를 제공한다. 이 Angular CLI는 Node.js 6.9.0, npm 3.0.0 이상을 필요로 한다.
